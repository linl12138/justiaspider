Spider
====

Scrapy & Python3

### Start

* `docker-compose up -d`
* `docker-compose exec spider sh` and you are free to go
* Update: `docker pull registry.cn-hangzhou.aliyuncs.com/rallets/spider`
* Build and Upload: `./bin/build.sh push`
